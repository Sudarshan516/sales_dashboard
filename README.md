# 📊 Sales Dashboard (Excel / BI Project)

This project showcases an interactive Sales Dashboard designed to analyze business performance using dynamic visualizations and user-friendly features.

 🚀 Key Features
 📈 KPIs Overview
Total Sales
Total Profit
Unique Orders
📊 Interactive Visualizations
Category-wise Sales Analysis
Region-wise Sales Distribution (Pie Chart)
Year-wise & Month-wise Sales Trends (Line Chart)
State-wise Sales Representation (Map Visualization)
 🎯 Filters & Slicers
Category filter (Furniture, Office Supplies, Technology)
Region filter (Central, East, South, West)
Time-based filtering (Month & Year)
 📥 Download Report Feature
Integrated Download PDF button to export live sales reports instantly.
#🔗 External Integration
Added a hyperlink to Flipkart Live Page for real-time reference and navigation.
 🛠️ Tools & Technologies Used
Microsoft Excel / Power BI
Data Cleaning & Transformation
Interactive Dashboard Design
Data Visualization Techniques
